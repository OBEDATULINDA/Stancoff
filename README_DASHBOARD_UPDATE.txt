IMPORTANT DATA RULE
The Suppliers > Location field is used as the coffee source area.
Use one consistent spelling for each area, for example:
Kisinga, Kasese
Kisinga, Kasese
not variations such as Kisinga / kisinga / KISINGA.

HOW TO UPDATE
1. Extract this ZIP.
2. Replace the matching files in your GitHub repository.
3. Keep the templates folder and static folder intact.
4. Commit and push.
5. On Render, choose Manual Deploy > Clear build cache & deploy.
